## Usage

To test script in the current shell environment, use `source`:

```bash
$ source ./file-name
```

Make script executable and run:

```bash
$ chmod +x file_name
$ ./file_name
```

## Requirements

* Ubuntu 20.04 LTS
* Bash shell
* Scripts should be executable and start with shebang `#!/bin/bash`
